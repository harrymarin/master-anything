# Master Anything

Master Anything is a universal learning skill for Codex/OpenClaw. It helps a user learn any field from zero to mastery through plain-language teaching, strategic learning maps, top-tier mental models, deliberate practice, and transfer-oriented coaching.

`master-anything` 是一个通用学习 skill，面向 Codex/OpenClaw 使用。它帮助用户把任何陌生领域从零基础推进到接近精通，方法包括白话讲解、战略学习地图、顶级思维模型、刻意练习和可迁移的陪练式推进。

## What It Does

- Starts from zero by default
- Explains in plain language before jargon
- Builds a strategic map before details
- Uses mental models to compress learning paths
- Forces practice, correction, and real application
- Pushes toward explanation, application, judgment, transfer, and teaching

## 它能做什么

- 默认按零基础带学
- 先讲人话，再讲术语
- 先给全局地图，再深入细节
- 用思维模型压缩学习路径
- 强制进入练习、纠错和真实应用
- 最终推进到能解释、能应用、能判断、能迁移、能教别人

## Install

Copy this folder into `~/.codex/skills/` or another compatible skills directory.

## 安装

把整个文件夹复制到 `~/.codex/skills/` 或兼容的 skills 目录即可。
